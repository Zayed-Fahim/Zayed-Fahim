## 🚀 About Me

I'm a backend-leaning full-stack engineer working as **SDE-I at [Shomvob Technologies](https://shomvob.com)**, where I build an **HRIS SaaS platform** — payroll engines, attendance & roster systems, leave management, and bulk employee onboarding — used by companies across Bangladesh.

- 🔭 **Currently:** designing REST APIs with **NestJS + TypeORM** on **PostgreSQL**, and background job pipelines with **BullMQ + Redis**
- 💬 **Ask me about:** NestJS, TypeORM, payroll & attendance domain modeling, React, Next.js
- 👯 **Open to:** collaborating on open-source TypeScript / Node.js projects
- 🌐 **Portfolio:** [zayedfahim.vercel.app](https://zayedfahim.vercel.app)
- ⚡ **Fun fact:** I go to sleep *after* having tea/coffee ☕
